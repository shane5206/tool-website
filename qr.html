什麼是 QR Code？

QR Code（Quick Response Code）是一種二維條碼，由日本 Denso Wave 於 1994 年發明，可儲存文字、網址、聯絡資訊等多種資料。只需用智慧型手機的相機掃描，即可立即開啟網頁、連接 WiFi 或儲存名片，是行銷、活動、標籤、包裝上最常見的資訊載體。

如何使用這個工具？

  1. 選擇模式：文字/網址（最通用）、WiFi 密碼（讓訪客掃一掃就連線）、名片 vCard（掃描後直接儲存聯絡人）。
  2. 填入對應資訊，QR Code 會即時出現在右側預覽。
  3. 調整尺寸與容錯等級（若要在 QR Code 上疊加 Logo，請選 H 級）。
  4. 點「↓ 下載 PNG」儲存高清圖檔，或「複製圖片」直接貼入簡報、文件。

FAQ

容錯等級要選哪一種？

一般用途選 M（15%）即可，在 QR Code 上印 Logo 或裝飾圖案時，請選 H（30%），即使圖案遮住部分格點仍能正確讀取。容錯越高，QR Code 越密集，儲存的資料量也略少。

QR Code 有效期限嗎？

本工具產生的是靜態 QR Code，將資料直接編碼在圖案中，沒有有效期限，也不需要伺服器。只要圖片完整、不模糊，就能永久掃描使用。若需要統計掃描次數或日後更換連結，則需使用「短網址服務」產生的動態 QR Code。

WiFi QR Code 支援哪些手機？

iOS 11 以上（iPhone 7 之後）與 Android 10 以上的原生相機 App 均支援掃描 WiFi QR Code 並自動連線。若手機較舊，可安裝任何 QR 掃描 App 來使用。

下載的 PNG 解析度足夠印刷嗎？

本工具最大可輸出 320 × 320 px 的 PNG，適合網頁、簡報、電子文件使用。若要印刷（名片、海報），建議選大尺寸並在印刷軟體中等比放大；QR Code 本身是向量式格點，放大後只要掃描區域清晰即可正確讀取，一般 A5 以下尺寸不成問題。

我的資料有上傳到伺服器嗎？

完全沒有。本工具為純前端運算，所有資料（網址、WiFi 密碼、名片資訊）均在您的瀏覽器中處理，不會傳送到任何伺服器，請放心使用。
