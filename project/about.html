關於工具網

輕量、原創、無干擾的線上工具集合。

我們在做什麼

「工具網」是一個專注於台灣使用者日常需求的線上工具集合。我們相信，許多日常瑣事——例如把數字轉成國字大寫、換算匯率、計算折扣——其實不需要安裝 App，也不需要註冊帳號，一個乾淨、夠快、答得準的網頁就夠了。

我們相信什麼

  • 原創優先。所有工具、文案、視覺皆為自家設計，不抄、不混淆。
  • 本機運算。能在瀏覽器算的事，就不送伺服器，不收集使用者輸入。
  • 速度即體驗。頁面輕量、即時顯示結果，不放冗長的開場動畫。
  • 適度廣告。使用 Google AdSense 維持營運，但保留合理閱讀區、不阻擋功能。

目前提供的工具

  • 數字轉國字大寫（新台幣）　— 目前主打工具。
  • 更多工具陸續推出中：日期轉換、匯率、折扣計算、QR Code 產生器⋯

有建議？

歡迎透過聯絡頁告訴我們你希望看到什麼工具，或回報任何錯誤。我們是兩人小團隊，每一封來信都會看。
